Repository van Diana Radueva



Structuur: De repository is goed gestructureerd want ik kan daar de nodige informatie zonder problemen vinden. Bijvoorbeeld zijn de bestanden voor de klant in een aparte map met een duidelijke naam ("klant"). Andere map, die "img" is, heeft een afbeelding voor de pagina van de zelfvoorstelling.
Volledigheid: Diana heeft 5 oefeningen toegevoegd: de logboek, de zelfvoorstelling, het interviewen van de klant, de stijl ("layout.css" bestand) en de hoofdpagina van de website voor deze klant.
Regelmatigheid : Diana plaatst updates (commits) regelmatig (twee keer per elke week). De laatste update was 6 november 2017.
Correctheid: In "Logboek.html" en "interviewKlant.html" moet er css elementen of een link naar een css pagina in de code toegevoegd worden. De bestand "layout.css" heeft html elementen zoals "h1", "p", "body", enz. maar deze elementen kunnen slechts in html bestanden gebruikt worden dus ze hoeven niet in css bestand te zijn. Maar dat is niet erg want er zullen nog de pagina's zoals "contact", "news", "home", en "about" toegevoegd worden.
Bereidheid tot verbetering: Diana heeft de inhoud toegevoegd in de bestanden van de map "Klant". In de bestanden van logboek en zelfvoorstelling heeft de studente niet alleen de inhoud toegevoegd maar ook deze gewijzigd en verbeterd.
